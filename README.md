# 마인크래프트 가챠 플러그인
...는 마인크래프트 플러그인 중 하나로 럭키블럭과 비슷한 기능을 수행하는 플러그인입니다.
서바이벌 모드로만 진행가능하며, 조금 더 재미를 느끼기는 데에는 부족함이 없다고 생각합니다.

보기전에 스타 한 번씩 눌러주시면 감사하겠습니다.
